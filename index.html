🌟 最新成果物：クリック・レストラン

【概要】 画面をクリックして料理を増やし、レストランを大きくしていくクッキークリッカー風の放置・クリック育成シミュレーションゲーム。
【こだわり】 スマホ画面でも操作自体は可能ですが、快適なクリック連打や大画面での演出を楽しめるよう「PC環境でのプレイ」を強く推奨しています。

  • 開発技術： HTML5 / CSS3 / JavaScript (ES6)
  • 公開先： GitHub Pages
